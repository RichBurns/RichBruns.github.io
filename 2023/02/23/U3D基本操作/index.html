U3D基本操作

Created2023-02-23|Updated2023-03-02|U3D
|Word count:361|Reading time:1min|Post View:

Scene面板控制

  • 鼠标右键 旋转
  • 鼠标滚轮 前进后退
  • 摁下滚轮 拖动
  • 选中f键 剧中
  • alt+鼠标左键 围绕旋转
  • 鼠标右键+wasd 场景漫游
  • ctrl+D 复制
  • ctrl+shift+f 定位物体到中心

贴合两个物体

  • 例如两个cube，选中其中一个按v键，拖动至参照点，在尝试贴合即可
  • 轴心点切换 Center/Pivot

Inspector面板

  • Transform
    Position世界位置 Rotation自身旋转

Material材质

  • rendering mod 渲染模式

    1. Opaque 不透明的
    2. cutout 剪纸，镂空，只显示不透明的
    3. Transparent 透明玻璃
    4. fade 褪色，淡入淡出

透明玻璃

现在我们需要制作一个玻璃墙：

  1. 创建一个cube
  2. 新建材质，并添置cube
  3. 选择rendering mod:Transparent
  4. 调整Albedo中A的值

空气墙

现在我们需要制作一个空气墙：

  1. 创建一个cube
  2. 新建材质，并添置cube
  3. 选择rendering mod:Fade
  4. 调整Albedo中A的值

Camera

Clear Flags 空白部分显示

  • Skybox 围绕整个场景的包装器，用于模拟天空

    1. six Sided 天空为六个面
    2. Procedural
    3. Depth Only 仅深度

    使用天空盒

    方式一：为摄像机添加组件

    方式二：光照窗口lighting-environment lighting-skybox

  • solid color

Culling Mask 选择遮罩、想要的图层

Projection 投射

  • Perspection 透视
  • Orthographic 正交

Clipping Planes

  • Near 只能看到x米以外的距离
  • Far 只能看x米的距离，例如为提高性能只渲染200m的物体
